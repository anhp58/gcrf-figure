How to access heroku

1. Install heroku CLI
2. Login to heroku with email
3. heroku access --app=gcrf-visualization
4. git status
5. git add .
6. git commit -m "a description of the changes"
7. git push heroku master

Update the code and redeploy in heroku

1. git status
2. git add.
3. git commit -m "a description of the changes"
4. git push heroku master

Update to git@github.com:anhp58/gcrf-figure.git

1. git checkout master
2. git remote add origin git@github.com:anhp58/gcrf-figure.git
3. git status
4. git add .
5. git commit -m "a description of the changes"
6. git push origin master
